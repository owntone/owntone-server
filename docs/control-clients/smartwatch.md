# Smartwatch Remote Control

[Pebble OwnTone Remote](https://apps.rebble.io/en_US/application/699cdba9484121000903f7d2)
is a Pebble watch app that lets you control OwnTone directly from your wrist.

## Features

- Now-playing info (title, artist, album) with playback controls
- Play/pause, next/previous track
- Volume control in Â±5% steps
- Voice search for playlists, artists, and albums
- Random playback by playlist, artist, or album
- Audio output management with per-output volume control
- Configurable favorites for frequently played content

## Getting started

Install OwnTone Remote from the [Rebble appstore](https://apps.rebble.io/en_US/application/699cdba9484121000903f7d2),
then open its settings in the Pebble app on your phone. Uses default OwnTone
server URL `http://owntone.local:3689`. Optionally configure favorites for quick
access to your most-used playlists, artists, and albums.

The source code is available on [GitHub](https://github.com/cdlenfert/pebble-owntone-remote).
